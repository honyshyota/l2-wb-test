Что выведет программа? Объяснить вывод программы.

```
package main
 
import (
    "fmt"
)
 
func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Вывод:

```
[77 78 79]
```

Создаем срез со второго эллемента по четвертый включительно из основного массива a
